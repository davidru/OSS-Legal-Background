## Key Themes Across US Case Law

1. **OSS licenses are enforceable.** Jacobsen v. Katzer + Artifex v. Hancom conclusively establish this.
2. **APIs have fair use protection.** Oracle v. Google provides strong (but not absolute) protection for API reimplementation.
3. **The AFC test dominates.** Computer Associates v. Altai's abstraction-filtration-comparison test is the standard framework.
4. **Derivative work scope is unresolved.** No court has decided the linking question in the GPL context.
5. **OSS is pro-competitive.** Wallace v. IBM confirms open source licensing promotes competition.
