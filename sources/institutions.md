# Key Institutions in Open Source Law

A directory of organizations whose publications, policy positions, and legal work are primary sources for open source software law.

> **Status:** Initial draft — contributions welcome.

## Standards & Licensing Bodies

| Institution | Focus | Key Publications |
|---|---|---|
| [Open Source Initiative (OSI)](https://opensource.org/) | Open Source Definition, license approval | OSD, approved license list, position papers |
| [Free Software Foundation (FSF)](https://www.fsf.org/) | Free software advocacy, GPL stewardship | GPL/LGPL/AGPL texts, license compliance guides |
| [Software Freedom Conservancy (SFC)](https://sfconservancy.org/) | GPL enforcement, fiscal sponsorship | Compliance guides, *Principles of Community-Oriented GPL Enforcement* |
| [Free Software Foundation Europe (FSFE)](https://fsfe.org/) | European free software policy | Legal & Licensing Workshop proceedings, REUSE initiative |
| [Creative Commons](https://creativecommons.org/) | Content licensing (adjacent to OSS) | CC license suite, license chooser |

## Foundations & Governance Organizations

| Institution | Focus | Key Publications |
|---|---|---|
| [Linux Foundation](https://www.linuxfoundation.org/) | Project hosting, governance frameworks | Joint Development Foundation, OpenChain, SPDX |
| [Apache Software Foundation](https://www.apache.org/) | Project governance, permissive licensing | Apache License 2.0, contributor guidelines, trademark policy |
| [Eclipse Foundation](https://www.eclipse.org/) | Open source governance (EU-based) | Eclipse Public License, specification process |
| [OpenJS Foundation](https://openjsf.org/) | JavaScript ecosystem governance | Project charters, IP policy |
| [CNCF](https://www.cncf.io/) | Cloud native ecosystem | Graduation criteria, IP policy |

## Academic & Research Institutions

| Institution | Focus | Key Publications |
|---|---|---|
| [Berkman Klein Center (Harvard)](https://cyber.harvard.edu/) | Internet & society, OSS governance | Research papers, Open Source Archetypes |
| [Stanford Center for Internet & Society](https://cyberlaw.stanford.edu/) | IP, open source, tech law | Fair use research, software freedom |
| [IFOSS Law Review / JOLTS](https://jolts.world/) | Dedicated OSS legal scholarship | Peer-reviewed articles on OSS law |
| [Max Planck Institute for Innovation](https://www.ip.mpg.de/) | Comparative IP law | FOSS licensing in civil law jurisdictions |
